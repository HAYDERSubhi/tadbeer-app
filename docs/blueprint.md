# **App Name**: Iraqi Budgeteer

## Core Features:

- Dashboard Overview: Displays a clear and concise summary of the user's budget, including income, expenses, and remaining balance.
- Voice Input with Iraqi Dialect Support: Utilizes speech recognition to allow users to verbally record expenses in Iraqi dialect. A tool analyzes the speech, extracts relevant information, and categorizes expenses automatically.
- Interactive Charts: Presents detailed visualizations of spending habits, budget allocation, and financial trends.
- Expense Categorization: Enables users to categorize their expenses based on predefined and custom categories.
- Personalization Settings: Provides options for users to customize the look and feel of the app, including theme and color options.

## Style Guidelines:

- Primary color: Keycard Yellow (#FFDA63).
- Secondary colors: Dark blues and grays.
- Accent: Teal (#008080) for interactive elements and data highlights.
- Clear, modern Arabic and Latin fonts for readability and a professional look.
- Consistent use of icons for categories and actions throughout the app.
- Right-to-left (RTL) layout support for Arabic text.
